Welcome to OpenVPN for Linux Clients



Processes Automated

  • Generating a Certificate Request with OpenSSL using shell scripts

  • Creating a Private key with OpenSSL using shell scripts

  • Having a Certificate Authority Sign your CSR and granting you a Certificate key

  • Creating your .ovpn file

Our Goal

This site is a project that allows potential clients to use OpenVPN (an open source VPN) to achieve secure point-to-point and site-to-site connections. The benefit of using this site is that the process of getting a certificate request signed by a Certificate Authority is automated and the client's public and private keys to be used are incorporated with the downloaded ovpn file.

How it Works

We use AES-128 for encryption keys and a combination of TLS and SSL protocols. OpenVPN utilizes UDP to transmit data for clients. You will be able to see in your configuration file that OpenVPN uses tls-auth for HMAC signature verification to avoid vulnerabilities like scanning and DoS attacks.

Getting Started

All you need to do to access your VPN is register an account, login, click the generate keys button and then download your .ovpn file. After your file is downloaded, we have a How To Use page for an easy to follow configuration guide for your .ovpn file.
